Bases del grid

Comandos utilizados para el ejercicio

  • grid-template-columns --> Define cuantas columnas se tendrá en el GRID.

  • grid-template-rows --> Define la altura de las filas que se crearan al definir las columns.

  • grid-auto-rows --> Define el tamaño de las filas que se generan automáticamente.

  • px - % - vw - auto - fr (fracción) --> Tipo de datos que soportan los atributos mencionados.

El poder de repeat

Esta función CSS representa un fragmento repetido de la lista de pistas , lo que permite escribir una gran cantidad de columnas o filas que exhiben un patrón recurrente en una forma más compacta

  • Ejemplo-1 --> Define de manera manual columns y rows, nada práctico si se tiene muchas columns y rows.

  • Ejemplo-2 --> Tenemos el mismo resultado del ejemplo-1 pero para aumentar las columns y rows cambiamos el numero de repeat

  • Ejemplo-3 --> Repeat puede ir al INICIO - MEDIO - FINAL al momento de declarar columns o rows como explica el código, soporta diferentes unidades (NO RECOMENDADO)

  • Ejemplo-4 --> Repite patrones en esta caso para las columns.
