Función repeat de grid

Esta función CSS representa un fragmento repetido de la lista de pistas , lo que permite escribir una gran cantidad de columnas o filas que exhiben un patrón recurrente en una forma más compacta

  • Ejemplo-1 --> Define de manera manual columns y rows, nada práctico si se tiene muchas columns y rows.

  • Ejemplo-2 --> Tenemos el mismo resultado del ejemplo-1 pero para aumentar las columns y rows cambiamos el numero de repeat

  • Ejemplo-3 --> Repeat puede ir al INICIO - MEDIO - FINAL al momento de declarar columns o rows como explica el código, soporta diferentes unidades (NO RECOMENDADO)

  • Ejemplo-4 --> Repite patrones en esta caso para las columns.

Función minmax de grid

Las funciones repeat y minmax de CSS Grid nos permiten jugar con los tamaños de las columnas y filas en la cuadrícula. En el caso de la propiedad minmax, esta nos permite establecer un tamaño mínimo y un tamaño máximo para las columnas o filas de nuestro grid. Y al final te evita de escribir @media(width > X){...}

  • Ejemplo-1 --> Se puede utilizar de manera independiente como en este caso. Define que la 1ra column no puede ser menor a 300px , en caso de que sea mayor se comportara como 1fr.

  • Ejemplo-2 --> Tambien es posible combinar con un repeat como muestra el ejemplo. La columna 3 y 4 no tendran un width menor a 150px , pero se comportara como 1fr si este es mayor a 150px

Ejemplo - 1

Ejemplo - 2


.gridMinMax1{
    box-sizing: border-box;
    display: grid;
    grid-template-columns: minmax(300px, 1fr) 1fr;
    padding: 10px;
}                

.gridMinMax2{
    box-sizing: border-box;
    display: grid;
    grid-template-columns: 1fr 1fr repeat(2, minmax(150px, 1fr));
    padding: 12px;
    column-gap: 5px;
    row-gap: 5px; 
}

Responsive con minmax y espacio entre cuadriculas con gap

Podremos hacer responsive las cuadriculas de manera fácil y utilizando el poder de grid sin crear media query, solo con minmax.

Column-gap, row-gap o simplemente gap es una propiedad abreviada shorthand para grid-row-gap y grid-column-gap ya que están obsoletas y que especifica los canales o separación entre las columns y rows de las cuadrículas.

  • Ejemplo-1 --> Con column-gap damos el espaciado entre cuadriculas solo a las columns, también se da un padding para crear el espacio entre el contenido y las cuadriculas Para lograr que el grid se comporte de manera responsiva declaramos las columns con repeat y el numero de columnas se generará con el auto-fit (Auto-ajusta el contenido si es que ya no tiene mas elementos y lo hace mas grande de ser necesario)

  • Ejemplo-2 --> Con row-gap damos el espaciado entre cuadriculas solo a los rows, también se da un padding para crear el espacio entre el contenido y las cuadriculas Para lograr que el grid se comporte de manera responsiva declaramos las columns con repeat y el numero de columnas se generará con el auto-fill (Auto-rellena con cuadriculas vacías si el grid ya no tiene mas elementos que mostrar)

  • Ejemplo-3 --> Y solo con gap se agrega el espaciado a las columns y rows al mismo tiempo

Ejemplo - 1

Ejemplo - 2

Ejemplo - 3

.gridGap1{
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(180px, 1fr));
    column-gap: 10px;
    padding: 10px;
}               
.gridGap2{
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(110px, 1fr));
    row-gap: 10px;
    padding: 10px;
} 
.gridGap3{
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(90px, 1fr));
    gap: 10px;
    padding: 10px;
}
